Sprint 0

Sign in / Create account with Gov Gateway and Defra ID

User group:

Supervising engineers

Our Assumptions

  1. Due to the sensitivity of the information being entered, our users will understand the importance of securely logging in to Government Gateway and Defra accounts.
  2. They will need to do this to either sign in or create an account to submit an S12 Statement online.

What we tested

  • how the user would go about searching for our service online
  • can they sign into our online service
  • how to create a Defra ID account

The scenarios

  1. User wants to register for a service, no Gov Gateway or Defra ID accounts
  2. User has Gov Gateway account and no Defra ID account
  3. Returning user has both Gov Gateway and Defra accounts - two factor authentcation
