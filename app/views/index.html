Submit supervising engineers statement for a reservoir

Beta MVP

These prototypes will test how a Supervised engineer can submit an S12 Statement online.

Sprint 0

We are are testing an assumption, that our users won't mind using Government Gateway and Defra ID to either sign in / create an account to submit an S12 Statement.

The three user scenarios we are testing are:

  1. User wants to register for a service, no Gov Gateway or Defra ID accounts
  2. User wants to register for a service, has Gov Gateway account no Defra ID account
  3. User has both Gov Gateway and Defra ID accounts authentication by sms or email
